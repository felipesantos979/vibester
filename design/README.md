# 🎨 Vibester - Design & UI/UX

Este diretório contém todos os artefatos visuais do aplicativo Vibester, projetados com foco em interface mobile-first, dark mode nativo (ideal para uso noturno).

## 📂 Estrutura de Arquivos
- `/assets`: Arquivos exportáveis (Logos, Mascote em SVG/PNG).
- `/mockups`: Telas exportadas do fluxo principal.
- `/styleguide`: Definições da paleta de cores e tipografia.
